Zoetrope - Image Viewer for LG3D

How to execute Zoetrope

Linux

  1. Copy build.xml and zoetrope.sh to lg3d-incubator directory.
  2. Go into lg3d-incubator directory
  3. Run the ant command 
    > ant sample-jar
  4. If the directory installed LG3D is not ../lg3d, edit zoetrope.sh and change LG3DHOME property.
  5. Run zoetrope.sh

Windows

  1. Copy build.xml and zoetrope.bat to lg3d-incubator directory.
  2. Go into lg3d-incubator directory.
  3. Run the ant command
    ant sample-jar
  4. If the directory installed LG3D is not ..\lg3d, edit zoetrope.bat and change LG3DHOME property.
  5. Run zoetrope.bat

Zoetrope reads images.txt in CLASSPATH, and determines the images files to display. Image files are not only usual files but also files included in JAR file.

Sample images.txt file is located in org/jdesktop/lg3d/apps/zoetrope/resources/images/sample, and there are some sample images in the same directory.

If there are images.txt in lg3d-incubator directory, Zoetrope reads it prior. The contents of images.txt is just list of image files.

In addition, you can run zoetrope.sh/zoetrope.bat with the argument:

> zoetrope.sh image-list.txt

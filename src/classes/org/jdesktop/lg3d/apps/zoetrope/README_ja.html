Zoetrope - Image Viewer for LG3D

実行方法

Linux の場合

  1. build.xml, zoetrope.sh を lg3d-incubator ディレクトリにコピーする。
  2. lg3d-incubator ディレクトリに移動
  3. ant を実行する
    > ant sample-jar
  4. LG3D がインストールされたディレクトリが ../lg3d でない場合、zoetrope.sh の LG3DHOME を編集して、正しいディレクトリを指すようにする
  5. zoetrope.sh を実行

Windows の場合

  1. build.xml, zoetrope.bat を lg3d-incubator ディレクトリにコピーする。
  2. lg3d-incubator ディレクトリに移動
  3. ant を実行する 
    ant sample-jar
  4. LG3D がインストールされたディレクトリが ..\lg3d でない場合、zoetrope.bat の LG3DHOME を編集して、正しいディレクトリを指すようにする
  5. zoetrope.bat を実行

Zoetrope はクラスパス中にある images.txt ファイルを読み込んで、表示するイメージを決定します。イメージは JAR ファイルに含まれるものでも、通常のファイルでもかまわないません。

サンプルの images.txt は org/jdesktop/lg3d/apps/zoetrope/resources/images/sample ディレクトリにあり、このディレクトリのイメージを表示するようになっています。

lg3d-incubator ディレクトリに images.txt が存在すれば、それを優先して読み込むので任意のイメージファイルを読み込むことが可能です。images.txt は単に画像ファイルを 1 行に 1 つづつ表記しただけです。

また、zoetrope.sh/zoetrope.bat の引数に読み込む画像ファイルの一覧を表したファイルを指定することもできます。

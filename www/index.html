Welcome to Project Looking Glass Incubator Project!

The Project Looking Glass incubator project is for community members to develop and share LG3D applications.

Use your imagination about what types of applications would be better in a 3D environment. You can see our music file manager as an example (The demo will show you an active version).

We invite you to develop 3D demonstration applications and host them here. Please refer to the Contributing to Project Looking Glass document to learn how to contribute to this project. Also, please see the JavaDesktop Community Governance Guidelines for general guidelines on participating in JavaDesktop community projects, such as Project Looking Glass.

To gain developer access to this project users simply need a signed JCA on file. Please pick a package name that does not conflict with other users of this project workspace and please ask permission from the owner of a package before committing any changes to their work.

As necessary incubator projects will be spun off into their own java.net projects.
